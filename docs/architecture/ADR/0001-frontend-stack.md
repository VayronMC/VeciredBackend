# ADR 0001: Lenguaje de Programación y Frameworks Frontend

* **Estado:** ✅ Aceptado

## Contexto
Se requiere definir las herramientas para la interfaz de usuario que permitan rapidez en el desarrollo y fidelidad con el diseño de Figma.

## Decisión
Se utilizará **JavaScript** como lenguaje base, **React.js** como framework principal, inicializado con **Vite**, y **Tailwind CSS** para los estilos.

## Justificación
* **JavaScript/React:** Permite trabajar mediante componentes reutilizables (como las tarjetas del tablón), reflejando perfectamente la estructura modular del diseño.
* **Vite:** Ofrece tiempos de compilación y recarga mucho más rápidos que otras alternativas.
* **Tailwind CSS:** Facilita la coincidencia exacta con la guía de estilos visuales aplicando clases directamente en el código.
